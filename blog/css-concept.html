Position Fixed

  • The element is fixed on the browser window which does not move when we scroll
  • Uses: Sidebars headers anything that needs to be constantly visible

    • Position absolute

  • Position absolute will make the element jump out of document flow, other elements will basically ignore it.
  • Uses: Tooltips, YouTube time stamps or promotional tags

    • Position relative

  • Position relative displays the element as normal but if position absolute is nested then it will be position relative to the elemnt rather than the page.
  • Uses: layering elements, creating overlapping effects
