Problem Solving

A time I solved being stuck on a simple problem.

The problem I was stuck on was completing the average function. I had console.log my average function to test that it works and yes it did, but when I tried using it in the getAverage function, the console.log output was undefined. This was confusing to me as everything looked correct and I had tested my average function.

After just staring at it for a few minutes I reached out to Shrena for help, and she pointed out that I wasn't returning anything in the getAverage function, for it to work I would need to return it. All I was missing was to return statement within getAverage function. I assumed that just returning the value inside the get average function would be enough, but I learned that each function that produces an output must have its own return statement.
